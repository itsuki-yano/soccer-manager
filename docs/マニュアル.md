## 1. アプリ概要

チームの試合・合宿にまつわる**配車費用の精算**や**備品管理**などを一元管理するアプリです。  
データはGoogleスプレッドシートに自動保存されるため、複数のマネージャーがリアルタイムで共有できます。

---

## 2. ホーム画面

アプリを開くと最初に表示されるメニュー画面です。

### メニュー一覧

| アイコン | 機能名 | 用途 |
|--------|--------|------|
| ⚽ | 試合・合宿管理 | 試合登録・配車当番の設定 |
| 🧃 | コーチ飲食費 | 飲み物代・食事代の管理 |
| 👟 | 選手マスタ | 選手・班・背番号の登録 |
| 📊 | Excel出力 | 精算書をダウンロード |

## 3. 試合・合宿管理

### 3-1. 試合一覧画面

#### 表示切り替え
- **一覧表示:** 日付順にカード表示（今後の予定 / 過去の試合）
- **カレンダー表示:** 月ごとのカレンダーで日程を確認

#### フィルター
- 種別（公式戦 / 合宿 / TM / その他）で絞り込み
- 「💴 精算あり のみ」で精算が必要な試合だけ表示

### 3-2. 手動で試合を登録する

「＋ 追加」ボタンをタップ。

| 入力項目 | 説明 |
|--------|------|
| 日付 | 試合日（必須） |
| 種別 | 公式戦 / 合宿 / TM / その他 |
| 試合名 | リーグ名など |
| 対戦相手 | チーム名 |
| 会場名 | グラウンド名など（必須） |
| 住所 | 入力すると距離を自動計算 |
| 精算あり | ONにすると精算書に反映 |

> **ポイント:** 住所を入力すると「かりがね小学校」からの往復距離が自動計算されます（約0.8秒後）。

### 3-3. BANDカレンダーから一括インポート

1. 一覧画面の「**BAND同期**」ボタンをタップ
2. BANDの予定が一覧表示される
3. 個別に「追加」、または「全てインポート」で一括登録

> ⚠️ **距離0km警告:** 住所から距離を計算できなかった場合、インポート後に警告が表示されます。「編集画面を開く」から住所を修正してください。

### 3-4. 試合詳細・編集

試合カードをタップすると詳細画面が開きます。

#### 配車当番の登録
1. 「配車当番を編集」ボタンをタップ
2. 選手一覧からドライバーを選択（班フィルターで絞り込み可能）
3. カスタム名を直接入力することも可能
4. 「保存」で確定 → 台数は自動的に当番人数に設定されます

#### 持ち帰り当番の登録
1. 「持ち帰り当番を編集」ボタンをタップ
2. 担当者を選択して保存

#### 距離の再計算
住所欄の横にある「再計算」ボタンをタップすると最新の距離に更新されます。

#### 精算ステータス
- **精算なし / 精算あり:** トグルで切り替え（精算書への反映）

#### 前回当番を引き継ぐ
直前の試合の配車当番をそのまま引き継げます（確認メッセージが表示されます）。

## 6. Excel出力（精算書）

配車費用の精算書を Excel ファイル（.xlsx）でダウンロードします。

### 出力前のプレビュー

画面上部に費用サマリーが表示されます。

| 表示項目 | 内容 |
|--------|------|
| 未請求 | まだ請求書を出していない試合 |
| 請求中 | 請求書を発行済みの試合 |
| 精算済み | 入金確認済みの試合 |

各試合のバッジ（未請求 / 請求中 / 精算済み）をタップすると、ステータスを循環して変更できます。

### 4種類の出力ボタン

#### 📋 請求中作成
- **対象:** ステータスが「請求中」の試合のみ
- **用途:** すでに請求書を出した試合の一覧を印刷・共有したいとき

#### 📤 未請求発行
- **動作:** 未請求の試合を全て「請求中」に変更してから出力
- **確認:** 何件変更するか確認画面が表示されます
- **用途:** 新しい請求書をまとめて発行するとき

#### ✅ 精算済発行
- **動作:** 期間を指定して「精算済み」の試合のみ出力
- **用途:** 精算が完了した試合を期間指定で集計・報告するとき

#### 📊 全て出力
- **動作:** 期間を指定して全ステータスの試合を3シートで出力
- **用途:** 全体の状況を一覧化したいとき

### 出力されるExcelの構成

| シート名 | 内容 |
|--------|------|
| 未請求 | 未請求の試合一覧と費用明細 |
| 請求中 | 請求中の試合一覧と費用明細 |
| 精算済み | 精算済みの試合一覧と費用明細 |
| 飲み物代請求 | コーチ飲食費の一覧 |

各シートには以下が含まれます：
- No. / 日付 / 曜日 / 対戦相手 / 会場 / 距離(km) / 1台費用 / 台数 / 合計費用 / 当番名
- 当番別の支払い明細（誰がいくら払うか）

> **費用計算式:** `往復距離(km) × ガソリン単価(円/km)` を10円単位で四捨五入

## 7. 備品管理

チームの備品（ボール・ビブス・救急セットなど）の在庫を管理します。

### 基本操作

- **追加:** 「＋ 追加」ボタン → 名称・数量・メモを入力
- **数量変更:** 「＋」「－」ボタンで増減（編集モード時）
- **写真登録:** カメラアイコンから写真を撮影・アップロード

### 編集モード

画面右上の「編集」ボタンで編集モードに切り替えると：
- 名称・メモのインライン編集
- 子アイテムの追加（入れ子構造で管理可能）
- 削除ボタンが表示される

### 入れ子構造

「救急セット」の中に「絆創膏」「消毒液」などを子アイテムとして登録できます。

## 8. 費用徴収管理

合宿費・クラブ費などの徴収状況を選手ごとに管理します。

### 費用の新規作成

「＋ 新規費用」から登録：

| 項目 | 説明 |
|-----|------|
| 費用名 | 「2024年合宿費」など |
| カテゴリ | 合宿費用 / クラブ費 / イベント費用 / その他 |
| 金額 | 1人あたりの金額 |
| 締切日 | 徴収期限 |
| 説明 | 備考 |

### 徴収状況の更新

費用詳細画面で各選手の「未納 / 徴収済」をタップして切り替えます。  
「全員を徴収済みにする」ボタンで一括更新も可能です。

### フィルター

- 班別フィルター（1班 / 2班 / …）
- 「未納のみ」表示で未納者だけ確認

## 9. 備忘録

連絡事項・メモを自由に記録できます。

- **作成:** テキストエリアに入力して「追加」
- **編集:** 各メモの「修正」ボタン
- **削除:** 「削除」ボタン（確認画面あり）
- **URLリンク:** メモ内に `https://...` を入力すると自動でリンクになります

## よくある質問

### Q. 距離が0kmになってしまう
住所が正確に入力されていない場合、距離を自動計算できません。  
試合詳細画面で「住所」欄に正確な住所（例：`愛知県豊田市○○町1-1`）を入力し、「再計算」をタップしてください。

### Q. BANDの予定が取得できない
BAND iCalのURLが正しく設定されているか確認してください（環境変数 `BAND_ICAL_URL`）。

### Q. 精算ステータスを間違えて変更してしまった
Excel出力画面のプレビューでステータスバッジをタップすると「未請求 → 請求中 → 精算済み → 未請求…」と循環して変更できます。

### Q. データはどこに保存されていますか？
Googleスプレッドシートに保存されています。設定画面の「スプレッドシートを開く」から直接確認できます。
